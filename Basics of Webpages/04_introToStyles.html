An Introduction to Styles

If you have seen all the files up until this point, you're probobly wanting to know the difference in styles between a div and a span. Before We go over that, let's start with what a style is.


A style is a but of CSS code which your HTML file will run to chagnge how it looks. there are three ways you can add CSS code to your HTML file

  1. You can create a CSS file and use the link tag in the head tag to link your CSS file to this HTML page.
  2. You can create a style tag in the head tag which can contain all of your CSS code.
  3. You can add styles as inline values to individual tags.

The last way is the worst way to add styles as you have to add styles to each tag individually unlike the first and second method where you can add styles to a bunch of like tags all at once (ex: adding a style to every p tag).

That being said, because we are trying to show each style indivually, it would be pretty annoying to tell you to switch between files or to get you to scroll up and down just to see what I've added to each tag so for this case, I will be adding each style on each tag individually but in a normal project, I recommend against it.

Let's start by walking you through how to add a style to a tag with the most basic and useful style (in my opinion). The background color.
