An Introduction to Styles

If you have seen all the files up until this point, you're probobly wanting to know the difference in styles between a div and a span. Before We go over that, let's start with what a style is.


A style is a but of CSS code which your HTML file will run to change how it looks. there are three ways you can add CSS code to your HTML file

  1. You can create a CSS file and use the link tag in the head tag to link your CSS file to this HTML page.
  2. You can create a style tag in the head tag which can contain all of your CSS code.
  3. You can add styles as inline values to individual tags.

The last way is the worst way to add styles as you have to add styles to each tag individually unlike the first and second method where you can add styles to a bunch of like tags all at once (ex: adding a style to every p tag).

That being said, because we are trying to show each style indivually, it would be pretty annoying to tell you to switch between files or to get you to scroll up and down just to see what I've added to each tag so for this case, I will be adding each style on each tag individually but in a normal project, I recommend against it.

Let's start by walking you through how to add a style to a tag with the most basic and useful style (in my opinion). The background color. (Also, to anybody who thinks I just spelt color wrong, if you spell color as "colour" when using CSS, it won't work. I'm from Canada where the correct spelling woudld be "colour" but make sure that you do spell it the way CSS wants or it'll do nothing).

In the tag you'd like add an inline style to, we'll start by creating the "style" value.

style

Then, we need to set it equal to a string of CSS code. Let's start by setting equal to an empty string.

style = ""

Now we can create some CSS code to put in the string. We can now set the background color. To change the background color, just type "background-color: " and then the color you want, followed with a semicolon (;). For example, if you wanted the background color to be cyan, we can type within the style string "background-color: cyan;".

style = "background-color: cyan;"

If we add this to our paragraph tag here, you'll see that the background of the paragraph tag is now cyan.

Now we're going to go into the "display" style


The display style is a little tricky to understand so we'll just start off with the basic two display types.

  • block (written as display: block)
  • inline (written as display: inline)

display block means that it takes the tag will be as wide as possible. This is the default for most tags which is why when you open the file, everything is on a different line. Because all the tags have a width of 100% of the screen, nothing else can be on the same line. This is also the reason why when we set the background color of the paragraph tags, the color will continue till it hits the edge of the screen.

display inline mean that it takes up as little space as possible. all characters for example would be considered inline because in order to have text to the right and to the left of each character, it needs to take up the exact amount of space needed for the chaaracter and nothing more. If characters instead had a display type of block, each character would be on a seprate line which is definitely not ideal.

we can now finally talk about the difference between a div tag and a span tag. It's quite simple actually.

  • div tag has a display of block and is meant for grouping block elements
  • span tag has a displau of inline and is meant for grouping inline elements

This explains why div tags are meant for grouping sevral tags and span tags are meant for grouping chunks of text. Because most tags are block elements by default and all text is act like inline elements. I'll show you how we can now use this to help us control the background color of different tags and text.
